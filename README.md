# 1、概览
#### 使用基于 stm32f103c8t6 制作的智能手表，所有基本要求与扩展要求均已完成，并通过测试

# 3、功能介绍
## 1、OLED主界面
### 1、时间显示
#### 1、当前时间
显示当前时间，可以与进入设置界面调时间

#### 2、日期
当日日期

#### 3、步数统计
显示当日走过步数，以及估算出来走过路程

### 2、传感器数据
#### 1、姿态角
仰视角
旋转角
航向角

#### 2、加速度
三个方向的加速度
#### 3、温度
当前温度

### 3、运动数据
#### 1、今日步数
显示今日步数

#### 2、运动距离
根据步数估算总运动距离

#### 3、卡路里
根据运动距离估算卡路里

### 4、设置
#### 1、时间设置
可以更改时间

#### 2、蓝牙配对

显示蓝牙配对的密码

#### 3、屏幕亮度
调节屏幕亮度

#### 4、闹钟设置
设置闹钟时间并设置什么时候开启
#### 5、回复出厂
回复上电后默认设置

### 5、系统信息
#### 1、电池电压
显示充电宝供电电压

#### 2、固件版本
当前固件版本，1.0.0

#### 3、蓝牙 MAC
显示手表蓝牙地址


## 2、各模块功能介绍
### 1、EC 11 旋转编码器
左右旋转移动光标位置，短按进入菜单，长按返回，在设置时间或设置闹钟界面下，短按会移动光标，旋转旋钮会改变时间设置

### 2、蓝牙 EC05 模块
与手机通讯，可以向手机发送步数等信息，手机可以通过发送 16 进制帧命令调手表信息

### 3、OLED 屏
显示交互界面，多级菜单

### 4、mpu 6050
用于得到各方向加速度等参数，并用于计算步数等衍生参数
